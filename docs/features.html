Tools

Utilities and helpers beyond the core marker → rig → skin workflow.

Detection Engines

Geometric

Reads mesh geometry directly. No training or AI, deterministic, works on stylized and realistic characters. Available in both editions.

AI (Full Edition)

Trained neural networks for more sophisticated placement — better on unusual proportions and extreme poses. Bundled, no setup required, and falls back to geometric detection if it can't load.

Viewport Helpers

  • Show / Hide Markers — toggles markers, connecting lines, and labels together
  • X-Ray Toggle — see through the mesh to markers and bones inside it
  • Mesh Selection Toggle — switch quickly between mesh and armature selection
  • Marker Resize — adjust the visual size of all markers
  • On-screen hints — contextual help text for the selected marker

Utilities

  • Check All Markers — validates missing, overlapping, or out-of-mesh markers
  • Fan Bones — generates helper bones at joints for smoother deformation
  • Straighten Fingers — removes side-to-side wander from finger placement
  • Mirror Weights — copies skin weights left↔right
  • Analyze / Repair Joint Bleed — finds and fixes areas where nearby geometry deforms incorrectly
  • Smooth Weights — blends weight transitions for more organic bending
  • Prune Weights — removes negligible weight values

Tips

  • Start with geometric detection — it's usually accurate enough and faster
  • Use Live Symmetry to save time on bilateral adjustments
  • Run "Check All Markers" before alignment
  • Test deformation in Pose Mode right after skinning to catch issues early
  • Paint weights in small batches and re-test frequently
  • Save a backup .blend before major modifications
