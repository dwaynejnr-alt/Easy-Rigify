Features & Tools

Easy Rigify includes a comprehensive set of tools beyond the core workflow. This page covers everything from viewport helpers to advanced utilities.

Detection Engines

Geometric Detection

Works on any humanoid mesh by reading its geometry.

  • No training or AI needed
  • Deterministic (same input = same output)
  • Works on stylized and realistic characters
  • Available in both Full and Lite editions

AI Detection (Full Edition)

Uses trained neural networks for more sophisticated placement.

  • Better on unusual proportions
  • Handles extreme poses (curled fingers, tucked thumb)
  • Bundled runtime, no setup required
  • Falls back to geometric if AI fails
💡 Which to use? Start with geometric (faster). If results aren't perfect, try AI detection for better accuracy on tricky characters.

Live Symmetry

What It Does

Edit one side of your character and see changes mirror in real time across the symmetry plane.

Enable Live Symmetry

  1. Go to the Markers tab
  2. Look for the "Live Symmetry" toggle in the Mirror section
  3. Check the box to enable
  4. Now when you move a marker, its mirror moves too

Use Cases

  • Fine-tuning bilateral symmetry
  • Fixing both sides of the body at once
  • Creating truly symmetrical characters
⚠️ Note: Disable Live Symmetry for asymmetrical characters (scars, deformities, etc.).

Viewport Helpers

Show / Hide Markers

Toggle all markers, connecting lines, and labels on/off.

  • Useful for seeing your mesh clearly during setup
  • Toggles markers and visual guides together

X-Ray Toggle

See through your mesh to view bones and markers inside.

  • Useful when markers are inside the character
  • Helps with alignment checking

Mesh Selection Toggle

Quickly switch between mesh and armature selection.

  • Grab a marker without deselecting the mesh
  • Switch between editing mesh and bones

Marker Resize

Adjust the visual size of all markers.

  • Make them bigger if they're hard to see
  • Make them smaller to reduce clutter

On-Screen Hints

Contextual help text appears in the viewport when you select a marker.

  • Shows exactly where each marker should go
  • Provides guidance for placement

Utility Tools

Check All Markers

Validates your marker setup before rigging.

Checks for:

  • Missing required markers
  • Overlapping markers
  • Markers outside the mesh

Fan Bones

Generate helper bones for smoother deformation at joints.

  • Creates extra deform bones at shoulders, elbows, etc.
  • Reduces creasing and pinching
  • Generated automatically during rig creation (optional)

Straighten Fingers

Cleans up finger placement to remove side-to-side wander.

  • Aligns fingers to a straight line
  • Useful if manual placement is crooked

Mirror Weights

Copy skin weights from left side to right (or vice versa).

  • Useful after editing one side
  • Maintains perfect symmetry

Analyze Joint Bleed

Finds areas where nearby geometry deforms incorrectly.

  • Visualizes problem areas in color
  • Helps you identify weight issues

Repair Joint Bleed

Automatically fixes joint bleed areas.

  • Restricts vertex influence to nearest joints
  • Reduces unwanted deformation

Smooth Weights

Smooths weight transitions across the mesh.

  • Removes hard edges in deformation
  • Creates more organic bending

Prune Weights

Removes tiny weight values that don't contribute.

  • Cleans up the weight data
  • Can improve deformation efficiency

Advanced Features

Backup & Restore Custom Changes

Save and restore custom rig modifications when regenerating.

  • Backup before tweaking the metarig
  • Regenerate the rig
  • Restore to reapply custom changes

Bone Roll Calculation

Automatic bone roll calculation from mesh geometry.

  • Ensures correct deformation direction
  • Based on surface normals
  • Can be manually adjusted if needed

Per-Finger Picker

Redo individual fingers without redoing the whole hand.

  • Select a finger and re-detect it
  • Great for fixing one problem digit

Section-by-Section Face Detection

Detect facial regions independently.

  • Eyes, brows, lips, nose detected separately
  • Gives you fine-grained control
  • Use separate mesh objects for better results

Pro Tips

  • Start with Geometric: It's usually accurate enough and faster
  • Use Live Symmetry: Saves time on bilateral adjustments
  • Validate Early: Run "Check All Markers" before alignment
  • Test Deformation: Pose the rig immediately after skinning to catch issues
  • Iterative Refinement: Paint weights in small batches and test frequently
  • Use Presets: "Default" weight tightening works for most characters
  • Keep Originals: Save a backup .blend before major modifications
