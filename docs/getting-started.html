Getting Started

Requirements

  • Blender 4.2 or newer — The addon uses features from Blender 4.2 and later.
  • Rigify addon enabled — Built into Blender. Enable it in Preferences → Add-ons.
  • No internet connection needed — Everything runs locally on your machine.
  • Humanoid mesh — Works on realistic scans, stylized characters, any scale.

Choosing Your Edition

Easy Rigify comes in two editions that work together seamlessly.

Body Detection (AI) ✅ Yes ❌ No
Body Detection (Geometric) ✅ Yes ✅ Yes
Finger Detection (AI) ✅ Yes ❌ No
Face Detection (AI) ✅ Yes ❌ No
Manual Marker Placement ✅ Yes ✅ Yes
Full Rigging Pipeline ✅ Yes ✅ Yes
Skinning Tools ✅ Yes ✅ Yes
Download Size ~50 MB ~1 MB
💡 Tip: Files saved in the Full Edition open in Lite and use the geometric engine. Both editions can't be installed at the same time, but your workflows are compatible.

Installation

Step 1: Download

Download the latest version from GitHub Releases.

Step 2: Install in Blender

  1. Open Blender
  2. Go to Edit → Preferences
  3. Click Get Extensions (Blender 4.2+)
  4. Click the dropdown and select Install from Disk...
  5. Browse to the downloaded zip file and select it
  6. Click Install Extension

Step 3: Enable the Addon

  1. In the Extensions list, search for "Easy Rigify"
  2. Check the box next to it to enable
  3. The AI runtime (Full Edition) will load automatically
  4. You'll see a new "Easy Rigify" tab in the 3D Viewport sidebar (press N)
ℹ️ Note: On first load, the full edition downloads the AI runtime (~100 MB). This only happens once. If the AI fails to load on your platform, the geometric detectors still work.

Quick Start (15 minutes)

1. Prepare Your Model

  • Apply all transforms: Object → Apply → All Transforms
  • Make sure your mesh is in a T-pose or A-pose
  • Select your character mesh in the 3D Viewport

2. Open Easy Rigify Panel

  • Press N to open the sidebar
  • Look for the "Easy Rigify" tab
  • You'll see four tabs: Markers, Rig, Skin, Tools

3. Place Markers

  • Set your character mesh as "Body Mesh" in the Markers tab
  • Click "Auto Detect Body" (geometric) or "EasyDetect Body" (AI, full edition)
  • Review the placed markers and nudge any that are off
  • Click "Check All Markers" to validate

4. Generate the Rig

  • Go to the "Rig" tab
  • Click "Align Rig to Markers"
  • Click "Generate Rig"
  • Your Rigify rig is now ready

5. Skin Your Mesh

  • Go to the "Skin" tab
  • Click "Bind Mesh" to create armature modifier
  • Click "Tighten Weights" to clean up deformation
  • Use "Paint Weights" to fine-tune problem areas
✅ Done! You now have a production-ready rigged character.
