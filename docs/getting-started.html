Getting Started

Requirements

  • Blender 4.2 or newer — the addon uses features from Blender 4.2 and later.
  • Rigify enabled — built into Blender; enable it in Preferences → Add-ons.
  • No internet connection needed — everything runs locally.
  • Humanoid mesh — realistic scans, stylized characters, any scale.

Editions

Files saved in the Full Edition open in Lite and use the geometric engine. Both editions can't be installed at the same time, but scenes are compatible.

Installation

1. Download

Get the latest version from GitHub Releases.

2. Install in Blender

  1. Open Blender
  2. Go to Edit → Preferences
  3. Click Get Extensions
  4. Open the dropdown and select Install from Disk...
  5. Browse to the downloaded zip and select it
  6. Click Install Extension

3. Enable

  1. Search for "Easy Rigify" in the Extensions list
  2. Check the box to enable it
  3. The AI runtime (Full Edition) loads automatically
  4. A new "Easy Rigify" tab appears in the 3D Viewport sidebar (press N)
On first load, the full edition downloads the AI runtime once. If it can't load on your platform, the geometric detectors still work.

Quick Start

1. Prepare your model

  • Apply all transforms: Object → Apply → All Transforms
  • Put the mesh in a T-pose or A-pose
  • Select the character mesh

2. Place markers

  • Press N, open the "Easy Rigify" tab, go to Markers
  • Set your mesh as "Body Mesh"
  • Click "Auto Detect Body" (geometric) or "EasyDetect Body" (AI)
  • Nudge any markers that are off, then click "Check All Markers"

3. Generate the rig

  • Go to the Rig tab
  • Click "Align Rig to Markers", then "Generate Rig"

4. Skin the mesh

  • Go to the Skin tab
  • Click "Bind Mesh"
  • Click "Tighten Weights" to clean up deformation
  • Use "Paint Weights" to fine-tune problem areas
That's a complete, riggable character. See the section pages below for details on each step.
