Key Features

🎯 One-Click Detection

Auto-place body markers for spine, arms, legs, and feet. AI detection handles tricky proportions in the full edition.

✋ Smart Fingers

Place every hand and finger joint in one click, or use the intuitive two-click-per-finger workflow (tip, then knuckle).

👁️ Facial Markers

Guided detection for brows, eyes, lids, lips, nose, cheeks, chin, jaw, and forehead. Full edition adds one-click detection.

🪞 Live Symmetry

Mirror markers left ↔ right. Turn on Live Symmetry to edit one side and mirror changes in real time.

⚙️ Auto-Align & Generate

Snap every metarig bone to your markers and generate the final Rigify rig, with correct bone rolls included.

🎨 Skinning Suite

Automatic binding, per-region weight tightening, joint-bleed repair, smoothing, and a full weight-paint toolkit.

Installation

1

Download

Get the latest version from GitHub Releases

2

Install in Blender

Edit → Preferences → Get Extensions → Install from Disk... → Pick the zip file

3

Enable & Done

Check the box next to "Easy Rigify" to enable. AI runtime loads automatically (full edition).

Requirements: Blender 4.2+, Rigify addon enabled. No internet or account needed.
