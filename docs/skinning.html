Skinning & Weight Cleanup

Skinning binds your mesh to the rig and defines how the mesh deforms when bones move. Easy Rigify provides automatic binding and comprehensive weight cleanup tools.

Automatic Binding

What is Binding?

Binding (also called "weight painting" setup) attaches your mesh to the rig by creating an Armature modifier and initial vertex weights.

Bind Your Mesh

  1. Select your character mesh
  2. Go to the Skin tab in Easy Rigify
  3. Click "Bind Mesh"
  4. Easy Rigify will:
    • Create an Armature modifier
    • Initialize weights based on bone proximity
    • Set up basic vertex group assignments
✅ Bound! Your mesh is now attached to the rig and will deform when you move bones.

Weight Cleanup

The Challenge

Automatic binding creates a good starting point, but weight deformation often needs refinement. Areas like shoulders, elbows, wrists, and knees can look lumpy or collapse.

Tighten Weights

Easy Rigify's weight tightener reduces unwanted deformation at joints.

Using Tighten Weights

  1. Click "Tighten Weights"
  2. Choose a preset:
    • Gentle: Subtle tightening, less aggressive
    • Default: Balanced for most characters
    • Aggressive: Strong tightening for exaggerated rigs
  3. Select which regions to process:
    • Fingers
    • Forearms / Upper Arms
    • Shoulders
    • Shins / Thighs
    • Spine / Neck
  4. Click "Tighten"
💡 Tip: Start with "Default" and test. If joints still look bad, try "Aggressive". For stylized rigs, "Gentle" might be better.

Fixing Common Deformation Issues

Shoulder Collapse

Symptom: Mesh sinks in when you raise the arm.

Solution:

  • Use "Tighten Weights" on the Shoulder region
  • Or manually paint weights on the shoulder area to reduce upper-arm influence

Elbow Pinching

Symptom: Forearm bends too sharply, looks creased.

Solution:

  • Use "Tighten Weights" on Forearms
  • Manually paint a smooth transition between forearm and upper-arm weights

Finger Deformation

Symptom: Fingers look twisted or collapse when bent.

Solution:

  • Use "Tighten Weights" on Fingers
  • Use "Smooth Weights" to blend vertex weights smoothly

Joint Bleed

Symptom: Nearby geometry deforms when it shouldn't (mesh from shoulder deforms with arm).

Solution:

  • Click "Analyze Joint Bleed" to find problem areas
  • Click "Repair Joint Bleed" to automatically fix them

Weight Tools

Smooth Weights

Smooths weight transitions across the mesh, reducing hard edges and creases.

  • Select the mesh
  • Click "Smooth Weights"
  • Adjust strength (1-10, higher = more smooth)

Mirror Weights

Copy weights from the left side to the right (or vice versa).

  • Go to Tools tab
  • Click "Mirror Weights"
  • Choose direction: L→R or R→L

Prune Weights

Remove very small weight values that don't contribute to deformation.

  • Click "Prune Weights"
  • Threshold: Remove influences below this value (default 0.01)

Weight Paint Entry/Exit

Toggle weight paint mode quickly.

  • Click "Enter Weight Paint" to start painting
  • Click "Exit Weight Paint" to return to Object Mode

Manual Weight Painting

When to Use It

Automated tools handle most cases, but some areas need hand-tuning:

  • Unusual character proportions
  • Stylized deformation (exaggerated secondary shapes)
  • Problem areas that tools couldn't fix

Basic Workflow

  1. Click "Enter Weight Paint" in the Tools tab
  2. Switch to Weight Paint Mode (top-right of 3D Viewport)
  3. Select a bone from the Outliner or armature
  4. Paint to increase/decrease influence on that bone
  5. Use the brush settings to adjust size and strength
  6. Click "Exit Weight Paint" when done
💡 Brush Tips:
  • Hold Shift to decrease weights (invert mode)
  • Use [ and ] to scale brush size up/down
  • Use lower Strength (opacity) for subtle adjustments

Testing Your Rig

Pose Mode Testing

  1. Select the rig armature
  2. Press Tab to enter Pose Mode
  3. Grab bones (G key) and move them to see deformation
  4. Look for problem areas (collapsing joints, excessive bleeding)

Common Test Poses

  • T-Pose: Raise arms straight out (should stay flat)
  • Bend Elbows: Flex arms (watch elbow crease)
  • Bend Knees: Squat down (watch kneecap and thigh roll)
  • Raise Shoulders: Shrug (should deform shoulder area only)
  • Close Fist: Flex fingers (watch hand and wrist)

Iterating

  1. Exit Pose Mode (Tab)
  2. Adjust weights using paint tools
  3. Return to Pose Mode and test again
  4. Repeat until satisfied
