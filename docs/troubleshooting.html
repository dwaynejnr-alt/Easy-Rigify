Troubleshooting & FAQ

Common Issues

Addon isn't showing in Blender

Edit → Preferences → Get Extensions → search "Easy Rigify" → make sure it's checked/enabled. Close and reopen Preferences; the tab appears in the 3D Viewport sidebar (N).

"Rigify addon not found"

Go to Edit → Preferences → Add-ons (not Get Extensions), search "Rigify", and enable it. Rigify ships with Blender but isn't on by default.

Markers won't place on my mesh

Make sure the mesh is selected and set as "Body Mesh". Apply all transforms (Object → Apply → All Transforms), check the mesh scale is reasonable (~1-2 Blender units tall), and make sure it's a single connected object.

Alignment is completely off

Usually unapplied scale/rotation. Delete the metarig, apply all transforms on your mesh, re-place and validate markers, then re-run "Align Rig to Markers".

AI detection doesn't work (Full Edition)

Check the system console (Help → Toggle System Console) for errors mentioning "onnxruntime" or "AI". Geometric detection works as a fallback if AI can't load on your platform.

Mesh deformation looks bad

Run "Tighten Weights" in the Skin tab, try a different preset, use "Analyze/Repair Joint Bleed", and manually paint problem joints with "Smooth Weights" to blend transitions.

Fingers look wrong or twisted

Use "Straighten Fingers", or re-detect a single finger with the per-finger picker. Make sure the hand is in a clear, neutral pose before detecting.

"Check All Markers" shows errors

Missing — place the marker at the position shown by the on-screen hint. Overlapping — move markers apart. Outside mesh — grab (G) and move the marker onto the mesh.

Blender crashes or freezes

Save first. Very high poly counts can cause slowdowns — try decimating. Update to the latest Blender 4.2+ and check the GitHub issues page for your version.

FAQ

Can I use Easy Rigify with other Rigify features?

Yes. It creates a standard Rigify rig — tweak control shapes, add custom controls, or use any other Rigify feature after generation.

Do both editions work together?

Yes. A file saved in the Full Edition opens in Lite using geometric detection. They can't be installed at the same time.

Can I use this for non-humanoid characters?

Not officially — it's built for bipeds with arms, legs, hands, and a face. Quadrupeds or alien anatomy need manual marker placement.

What's the maximum mesh size?

No hard limit, but detection is slower on very high poly counts. Consider decimating for faster detection.

Can I export the rig to game engines?

Yes — export as FBX or glTF. The Rigify rig is a standard armature and works with any engine supporting skeletal animation.

Is internet needed?

No. Everything runs locally, including the Full Edition's bundled AI runtime.

How do I report a bug?

Open an issue on GitHub with your Blender version, steps to reproduce, and a screenshot or error message.
